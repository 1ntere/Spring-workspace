2. thymeleaf

	웹 및 독립 실행형 환경 모두를 위한 최신 서버 Java 템플릿 엔진
	웹 실행 : 요청시 포워드 되는 화면
	독립 실행 : html 파일 실행(인터넷 연결 x)
	
	HTML 파일에서 th(ThymeLeaf)속성을 이용해
	컨트롤러부터 전달받은 데이터를 이용해 동적 페이지를 만들 수 있음
	
	Springboot에서는 jsp가 아닌 ThymeLeaf 사용을 권장하고 있음
	
	사용법
	HTML에서
	< !DOCTYPE html> 밑에 작성된
	< html xmlns:th="http://www.thymeleaf.org">작성 후 사용이 가능
